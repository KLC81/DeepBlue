# DeepBlue - STM32F103 多功能控制系统

## 项目概述
基于正点原子 STM32F103 精英版开发板的多功能控制系统，集成：
- 🔌 **16路继电器控制**（通过8路光耦隔离继电器模块×2）
- ⚙️ **步进电机驱动**
- 📺 **7寸 LCD 触摸屏界面**
- ⏰ **定时控制功能**
- 📱 **手机远程操作**（WiFi/蓝牙）

## 硬件配置
- **MCU**: STM32F103ZE
- **屏幕**: 7寸 LCD 触摸屏
- **继电器**: 8路光耦隔离继电器模块（×2）
- **步进电机**: NEMA系列
- **开发环境**: STM32CubeIDE
- **编程语言**: C

## 项目结构

DeepBlue/
