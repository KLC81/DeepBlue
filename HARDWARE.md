# Hardware Design Documentation

## Introduction

## Components
### 1. Microcontroller
- **Model**: Raspberry Pi 4
- **Specifications**: Quad-core CPU, 1GB RAM.
- **Purpose**: Acts as the main control unit.

### 2. Sensors
- **Type**: Ultrasonic sensors.
- **Purpose**: Provide distance measurements for navigation.

### 3. Actuators
- **Type**: Servo motors.
- **Purpose**: Responsible for movement and manipulation.
