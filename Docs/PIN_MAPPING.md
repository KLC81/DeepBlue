# Pin Mapping Documentation

This document provides details of the pin mapping for the DeepBlue project.

| Pin Number | Function          | Description                   |
|------------|-------------------|-------------------------------|
| 1          | VCC               | Power supply                  |
| 2          | GND               | Ground connection             |
| 3          | GPIO 0            | General purpose I/O pin 0    |
| 4          | GPIO 1            | General purpose I/O pin 1    |
| 5          | PWM               | Pulse-width modulation output  |
| 6          | ADC 0             | Analog to Digital Converter 0 |

## Notes
- Ensure to check the voltage levels for each pin.
- Consult the project schematics for more details.
